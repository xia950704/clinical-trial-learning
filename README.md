# 🎓 临床试验数据分析教程（中文）

> A practical, hands-on tutorial for clinical trial data analysis in Chinese.
> 面向中文读者的临床试验数据分析实战教程，从方案设计到统计输出。

---

## 📖 教程结构

本教程按**研究者发起临床试验（IIT）**的真实工作流程组织，每个阶段配有可运行的 R 代码示例。

```
clinical-trial-learning/
├── 01-试验设计/                    # Study Design
│   ├── Phase-I-II-设计.md          # Phase I/II 设计要点
│   ├── 随机化与样本量.md            # 随机化方法 + 样本量计算
│   └── 中期分析与组序贯.md          # 中期分析 + 组序贯设计
│
├── 02-数据管理/                    # Data Management
│   ├── 数据清洗-S1-去重.R           # 重复记录检测与去重
│   ├── 数据清洗-S2-日期标准化.R     # 日期格式统一
│   ├── 数据清洗-S3-分类变量标准化.R # 分类变量编码统一
│   ├── 数据清洗-S4-逻辑核查.R       # 逻辑一致性检查
│   └── 数据清洗-S5-缺失值处理.R     # 缺失值填补策略
│
├── 03-统计分析/                    # Statistical Analysis
│   ├── 描述性统计.R                 # Table 1 基线特征表
│   ├── 疗效分析.R                   # 主要终点分析（t检验/卡方/KM）
│   ├── 安全性分析.R                 # AE/SAE 汇总分析
│   └── 混合效应模型.R               # 重复测量数据（动物实验）
│
├── 04-TLF规范/                     # Tables, Listings, Figures
│   ├── 基线表-模板.md               # Table 1 模板
│   ├── 安全性表-模板.md             # 安全性汇总表
│   ├── 疗效表-模板.md               # 疗效汇总表
│   └── 图表生成.R                   # ggplot2 出版级图表
│
├── 05-Protocol撰写/                # Protocol Writing
│   ├── Protocol-中文模板.md         # SPIRIT 格式中文模板
│   └── Protocol-英文模板.md         # medRxiv 格式英文模板
│
└── 06-实战案例/                    # Case Studies
    ├── 尼非卡兰-IIT/                # 156例多中心RCT（真实项目）
    │   ├── 方案设计.md
    │   ├── 数据分析.R
    │   └── 统计结果.md
    └── 动物实验-肿瘤模型/           # 小鼠肿瘤体积分析（模板）
        ├── 分析模板.R
        └── 输出示例/
```

## 🚀 快速开始

### 环境准备

```r
# 安装依赖包
install.packages(c("tidyverse", "lme4", "ggpubr", "survival", "showtext"))
```

### 第一个分析（30分钟）

```r
# 运行动物实验模板
source("06-实战案例/动物实验-肿瘤模型/分析模板.R")
```

输出：出版级折线图 + 统计结果汇总
