# 🎓 临床试验数据分析学习笔记

> **用中文讲透 CDISC 标准，用方法学构建护城河**

---

## 📖 仓库介绍

本仓库是一个**专注临床试验数据分析**的个人学习平台，记录从 SDTM 到 ADaM、从试验设计到 TLF 生成的完整学习过程。

### 核心理念

| 维度 | 差异化定位 |
|------|-----------|
| **语言** | 全中文内容，消除语言障碍，让 CDISC 标准真正"落地" |
| **教学** | 不止于代码，更关注"为什么"与"怎么做"的思维过程 |
| **深度** | 拒绝碎片化，每个主题都提供方法学层面的系统梳理 |

### 内容范围

- 🔬 **CDISC 标准** — SDTM、ADaM、define.xml 详解
- 📊 **试验设计** — Phase I/II/III、随机化、样本量、中期分析
- 📈 **TLF 生成** — 基线表、安全性表、疗效表、生存分析
- 🧠 **方法学** — 因果推断、目标试验模拟、真实世界证据

### 第一步：环境准备

