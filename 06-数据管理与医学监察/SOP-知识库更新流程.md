## 📊 阶段详情

### 阶段 1：资源发现

| 项目 | 说明 |
|------|------|
| **目标** | 搜索 GitHub 公开临床试验资源，评估质量 |
| **输入** | 研究主题（如"肿瘤学临床试验"） |
| **输出** | 资源清单 + 质量评估报告 |
| **输出位置** | `11-临床试验学习/05-资源清单/GitHub资源.md` |
| **耗时** | 1-2 小时 |
| **关键动作** | 1. 搜索 GitHub（关键词：CDISC、ADaM、SDTM、clinical trial design）<br>2. 评估仓库质量（stars、更新频率、文档完整性）<br>3. 筛选 P0/P1/P2 优先级<br>4. 记录资源链接和关键文件 |

#### 资源评估标准

| 维度 | 评分标准 |
|------|---------|
| **质量** | ⭐⭐⭐⭐⭐（5 星）：完整文档 + 可运行代码<br>⭐⭐⭐⭐（4 星）：文档完整 + 部分代码<br>⭐⭐⭐（3 星）：有代码但文档不全<br>⭐⭐（2 星）：代码可运行但无文档<br>⭐（1 星）：仅示例代码 |
| **相关性** | 高：直接匹配学习主题<br>中：部分匹配<br>低：不匹配 |
| **可复现性** | 高：一键运行<br>中：需要配置<br>低：代码不完整 |

#### 输出模板

```markdown
# GitHub 资源清单 — [主题]

## P0 资源（核心学习）

| 资源 | ⭐ | 语言 | 学习重点 | 链接 |
|------|-----|------|---------|------|
| [资源名] | 5 | R | [内容] | [链接] |

## P1 资源（参考）

| 资源 | ⭐ | 语言 | 学习重点 | 链接 |
|------|-----|------|---------|------|

## P2 资源（扩展）

| 资源 | ⭐ | 语言 | 学习重点 | 链接 |
|------|-----|------|---------|------|
```

### 阶段 2：数据验证

| 项目 | 说明 |
|------|------|
| **目标** | 验证数据完整性和分析可复现性，检查知识重叠 |
| **输入** | 阶段 1 筛选的资源 |
| **输出** | 验证报告 |
| **输出位置** | `11-临床试验学习/06-实战项目/验证报告-[日期].md` |
| **耗时** | 2-4 小时 |
| **关键动作** | 1. 克隆 GitHub 仓库<br>2. 检查数据文件完整性（SDTM/ADaM 域是否齐全）<br>3. 尝试运行代码（验证可复现性）<br>4. 检查与现有 vault 内容重叠<br>5. 记录验证结果和潜在问题 |

#### 验证检查清单

| 检查项 | 说明 |
|--------|------|
| **数据完整性** | SDTM 域是否齐全（DM、AE、VS、LB、EX 等） |
| **代码可运行** | R 脚本是否能成功运行 |
| **文档完整性** | README、specs.xlsx、CRF 是否齐全 |
| **知识重叠** | 是否与 04-方法学与工具/ 内容重叠 |
| **依赖包** | 需要的 R 包是否可安装 |

| 项目 | 说明 |
|------|------|
| **目标** | 提取试验设计要素（终点、样本量、随机化等） |

#### 方案设计模板要素

| 要素 | 说明 |
|------|------|
| **研究类型** | Phase I/II/III、优效/非劣效/等效 |
| **主要终点** | 主要疗效终点（如 OS、PFS、ORR） |
| **次要终点** | 次要疗效终点、安全性终点 |
| **样本量** | 总样本量、各臂样本量、脱落率 |
| **随机化** | 随机化方法（简单/区组/分层）、分层因素 |
| **盲法** | 开放/单盲/双盲 |
| **中期分析** | 是否有中期分析、停止规则 |
| **统计假设** | H0、H1、alpha、beta |

### 阶段 4：统计学学习

| 项目 | 说明 |
|------|------|
| **目标** | 提取统计方法（假设检验、生存分析、多因素等） |
| **输入** | 阶段 3 提取的设计要素 |
| **输出** | 统计学方法库 |
| **输出位置** | `11-临床试验学习/05-资源清单/统计学方法库.md` |
| **耗时** | 3-5 小时 |
| **关键动作** | 1. 阅读统计分析计划（SAP）<br>2. 提取统计方法（假设检验、生存分析、多因素模型）<br>3. 记录 R 代码实现<br>4. 创建方法库文档 |

#### 统计学方法库要素

| 方法类别 | 具体方法 |
|---------|---------|
| **描述统计** | 均值、中位数、标准差、频数、百分比 |
| **假设检验** | t 检验、卡方检验、Fisher 精确检验、Wilcoxon 检验 |
| **生存分析** | Kaplan-Meier、Log-rank 检验、Cox 模型 |
| **多因素分析** | 线性回归、Logistic 回归、Cox 回归 |
| **因果推断** | IPTW、G-computation、BART、倾向评分匹配 |
| **多重比较** | Bonferroni、Holm、FDR |

#### 输出模板
