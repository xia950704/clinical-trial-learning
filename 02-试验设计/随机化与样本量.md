## 1. 随机化方法

### 1.1 简单随机化（Simple Randomization）

**定义**：每个受试者被分配到任一组的概率相等（通常为 1:1），类似于抛硬币。

**实现方式**：
- 使用随机数生成器或随机数表
- 每个受试者独立分配，不受之前分配结果影响

**优点**：
- 操作简单，易于实施
- 理论上可保证组间基线特征均衡（大样本下）

**缺点**：
- 小样本时可能出现组间人数不平衡
- 无法控制已知混杂因素的分布

**适用场景**：大样本试验（通常 n > 200）

---

### 1.2 区组随机化（Block Randomization）

**定义**：将受试者分成若干区组（block），每个区组内各组的分配人数固定，保证区组结束时组间人数平衡。

**示例**：区组长度 = 4，1:1 分配

| 区组 | 可能的排列（共 6 种） |
|------|----------------------|
| 1    | A A B B              |
| 2    | A B A B              |
| 3    | B B A A              |
| ...  | ...                  |

**优点**：
- 确保试验过程中任意时间点组间人数基本平衡
- 可防止时间趋势偏倚

**缺点**：
- 区组长度固定时，最后两个受试者的分组可能被预测（选择偏倚风险）
- 解决方案：使用**可变区组长度**（如 4、6、8 随机混合）

**适用场景**：多中心试验、需要持续入组的长期试验

---

### 1.3 分层随机化（Stratified Randomization）

**定义**：先按重要预后因素（如年龄、性别、疾病分期）分层，每层内独立进行区组随机化。

**步骤**：
1. 确定关键预后因素（通常不超过 3-4 个）
2. 每个因素分为若干层（如年龄：<60, ≥60）
3. 每层建立独立的随机化序列

**示例**：2 因素 × 2 水平 = 4 层

| 层 | 年龄 | 性别 | 随机化序列 |
|----|------|------|-----------|
| 1  | <60  | 男   | 区组随机化 |
| 2  | <60  | 女   | 区组随机化 |
| 3  | ≥60  | 男   | 区组随机化 |
| 4  | ≥60  | 女   | 区组随机化 |

**优点**：
- 确保关键预后因素在组间均衡分布
- 提高统计检验效能

**缺点**：
- 分层过多会导致每层样本量过小，失去均衡意义
- 实施复杂，需要 IT 系统支持

**适用场景**：已知强预后因素的小-中样本试验

---

### 1.4 动态随机化 / 最小化法（Dynamic Randomization / Minimization）

**定义**：每个新受试者入组时，根据已入组受试者的基线特征分布，计算将其分配到各组后各组间的"不平衡度"，分配到使不平衡度最小化的组。

**核心思想**：
$$\text{D}_j = \sum_{k=1}^{K} w_k \cdot |n_{j,k} - \bar{n}_k|$$
其中 $D_j$ 为分配到组 $j$ 的不平衡度，$w_k$ 为第 $k$ 个因素的权重。

**步骤**：
1. 新受试者基线特征输入系统
2. 计算分配到 A 组和 B 组的不平衡度
3. 分配到不平衡度较小的组（可加入随机成分，如 80% 概率选最优组，20% 随机）

**优点**：
- 可在多个预后因素上实现近乎完美的均衡
- 不受分层数量限制

**缺点**：
- 完全确定性时存在选择偏倚风险（必须加入随机成分）
- 需要实时计算系统支持
- 统计分析需考虑动态随机化设计（通常仍可用常规方法）

**适用场景**：多中心、多预后因素、中小样本试验

## 2. 样本量计算逻辑

### 2.1 三类假设检验框架

| 类型 | 零假设 $H_0$ | 备择假设 $H_1$ | 目的 |
|------|-------------|---------------|------|
| **优效试验** (Superiority) | $\mu_T - \mu_C \leq 0$ | $\mu_T - \mu_C > 0$ | 证明试验组优于对照组 |
| **非劣效试验** (Non-inferiority) | $\mu_T - \mu_C \leq -\delta$ | $\mu_T - \mu_C > -\delta$ | 证明试验组不劣于对照组（差值不超过 $\delta$） |
| **等效试验** (Equivalence) | $|\mu_T - \mu_C| \geq \delta$ | $|\mu_T - \mu_C| < \delta$ | 证明两组差异在等效界值 $\delta$ 内 |

> **关键区别**：优效试验是单侧检验，非劣效也是单侧但方向相反，等效试验是双侧检验（需同时证明不优也不劣）。

---

### 2.2 优效试验样本量逻辑

**目标**：检测出试验组与对照组之间至少 $\Delta$ 的真实差异。

**参数**：
- $\alpha$：I 类错误率（单侧，通常 0.025）
- $\beta$：II 类错误率（通常 0.1 或 0.2，对应效能 $1-\beta = 80\%$ 或 $90\%$）
- $\Delta$：临床有意义的差异（效应量）
- $\sigma$：总体标准差（连续变量）

**基本逻辑**：
$$n = \frac{2\sigma^2 (Z_{1-\alpha} + Z_{1-\beta})^2}{\Delta^2}$$

---

### 2.3 非劣效试验样本量逻辑

**目标**：证明试验组效果不比对照组差超过 $\delta_{NI}$（非劣效界值）。

**非劣效界值确定原则**：
- $\delta_{NI}$ 必须小于对照组与安慰剂的历史差异（保留部分疗效）
- 通常取历史效应量的 50%（"50% 规则"）
- 需监管机构认可

**样本量公式（连续终点）**：
$$n = \frac{2\sigma^2 (Z_{1-\alpha} + Z_{1-\beta})^2}{(\delta_{NI} - \Delta)^2}$$
其中 $\Delta$ 为预期真实差异（通常假设为 0，即两组效果相同）。

**注意**：分母为 $(\delta_{NI} - \Delta)^2$，当假设两组等效时 $\Delta = 0$，分母为 $\delta_{NI}^2$。非劣效试验通常需要**更大样本量**，因为 $\delta_{NI}$ 通常远小于优效试验的 $\Delta$。

---

### 2.4 等效试验样本量逻辑

**目标**：证明两组差异在 $[-\delta_E, +\delta_E]$ 范围内。

**统计方法**：双单侧检验（TOST, Two One-Sided Tests）
- $H_{01}: \mu_T - \mu_C \leq -\delta_E$ vs $H_{11}: \mu_T - \mu_C > -\delta_E$
- $H_{02}: \mu_T - \mu_C \geq +\delta_E$ vs $H_{12}: \mu_T - \mu_C < +\delta_E$
- 两个检验均拒绝 $H_0$ 才能得出等效结论

**样本量公式（连续终点）**：
$$n = \frac{2\sigma^2 (Z_{1-\alpha} + Z_{1-\beta/2})^2}{(\delta_E - |\Delta|)^2}$$

**注意**：
- $\alpha$ 通常取 0.05（双侧）
- 等效试验样本量通常最大，因为需要同时排除两个方向的差异

## 3. 样本量公式推导

### 3.1 均值比较（两独立样本）

#### 3.1.1 优效检验推导

**假设**：
- $H_0: \mu_T - \mu_C \leq 0$
- $H_1: \mu_T - \mu_C = \Delta > 0$
- 两组方差相等：$\sigma_T^2 = \sigma_C^2 = \sigma^2$
- 样本量相等：$n_T = n_C = n$

**检验统计量**：
$$Z = \frac{\bar{X}_T - \bar{X}_C}{\sqrt{\frac{2\sigma^2}{n}}} \sim N\left(\frac{\Delta}{\sqrt{2\sigma^2/n}}, 1\right) \quad \text{(在 } H_1 \text{ 下)}$$

**I 类错误控制**：
$$P\left(Z > Z_{1-\alpha} \mid H_0\right) = \alpha \Rightarrow \text{拒绝域边界为 } Z_{1-\alpha}$$

**效能要求**：
$$P\left(Z > Z_{1-\alpha} \mid H_1\right) = 1 - \beta$$

在 $H_1$ 下，$Z$ 的均值为 $\frac{\Delta}{\sqrt{2\sigma^2/n}}$，因此：
$$Z_{1-\alpha} = \frac{\Delta}{\sqrt{2\sigma^2/n}} - Z_{1-\beta}$$

解出 $n$：
$$\boxed{n = \frac{2\sigma^2 (Z_{1-\alpha} + Z_{1-\beta})^2}{\Delta^2}}$$

**Cohen's d 效应量形式**：
$$n = \frac{2 (Z_{1-\alpha} + Z_{1-\beta})^2}{d^2}, \quad d = \frac{\Delta}{\sigma}$$

---

#### 3.1.2 比例比较（两独立样本）

**假设**：
- $p_T$：试验组事件发生率
- $p_C$：对照组事件发生率
- $p = \frac{p_T + p_C}{2}$（合并比例）

**优效检验公式**：
$$\boxed{n = \frac{\left[Z_{1-\alpha}\sqrt{2p(1-p)} + Z_{1-\beta}\sqrt{p_T(1-p_T) + p_C(1-p_C)}\right]^2}{(p_T - p_C)^2}}$$

**简化公式**（当 $p_T \approx p_C \approx p$ 时）：
$$n \approx \frac{(Z_{1-\alpha} + Z_{1-\beta})^2 \cdot 2p(1-p)}{(p_T - p_C)^2}$$

**非劣效比例检验**：
$$n = \frac{\left[Z_{1-\alpha}\sqrt{p_T(1-p_T) + p_C(1-p_C)} + Z_{1-\beta}\sqrt{p_T(1-p_T) + p_C(1-p_C)}\right]^2}{(\delta_{NI} - (p_T - p_C))^2}$$

---

### 3.2 生存分析（Log-rank 检验）

#### 3.2.1 Schoenfeld 公式（Cox 模型）

**核心思想**：样本量取决于**事件数**而非总样本量。

**事件数公式**：
$$\boxed{E = \frac{4 (Z_{1-\alpha/2} + Z_{1-\beta})^2}{(\log HR)^2}}$$

其中 $HR$ 为风险比（Hazard Ratio）。

**总样本量**（考虑失访和事件发生率）：
$$N = \frac{E}{P(\text{event})} = \frac{E}{1 - S(t_{\text{follow-up}})^{1/2}}$$

更精确的公式（考虑入组时间 $a$ 和随访时间 $f$）：

$$N = \frac{E}{1 - \frac{1}{a}\int_0^a S(t)^{1/2} dt}$$

**简化版本**（均匀入组，指数生存）：
$$N = \frac{E}{1 - \frac{1 - e^{-\lambda f}}{\lambda a}}$$

其中 $\lambda$ 为对照组的事件发生率。

---

#### 3.2.2 非劣效生存试验

**假设**：
- $H_0: HR \geq HR_{NI}$（试验组风险不低于对照组的 $HR_{NI}$ 倍）
- $H_1: HR < HR_{NI}$

**事件数公式**：
$$E = \frac{(Z_{1-\alpha} + Z_{1-\beta})^2}{\left[\log(HR) - \log(HR_{NI})\right]^2}$$

当假设 $HR = 1$（两组真实效果相同）：
$$E = \frac{(Z_{1-\alpha} + Z_{1-\beta})^2}{[\log(HR_{NI})]^2}$$

---

### 3.3 公式参数速查表

| 参数 | 符号 | 常用值 | 说明 |
|------|------|--------|------|
| I 类错误（单侧） | $\alpha$ | 0.025 | 优效/非劣效 |
| I 类错误（双侧） | $\alpha$ | 0.05 | 等效试验 |
| II 类错误 | $\beta$ | 0.10 或 0.20 | 效能 80% 或 90% |
| $Z_{1-\alpha}$ | — | 1.96（双侧0.05）/ 1.645（单侧0.05） | 标准正态分位数 |
| $Z_{1-\beta}$ | — | 0.84（80%）/ 1.28（90%） | 标准正态分位数 |
| 效应量 | $\Delta$ 或 $d$ | 临床定义 | 最小临床有意义差异 |
| 非劣效界值 | $\delta_{NI}$ | 历史效应 × 0.5 | 需预先确定 |

### 4.3 survival 包 — 生存分析样本量

```r
library(survival)

# 方法1：手动计算 Schoenfeld 公式
calc_schoenfeld <- function(hr, alpha = 0.025, power = 0.8, two_sided = FALSE) {
  z_alpha <- qnorm(1 - alpha)
  z_beta <- qnorm(power)
  
  if (two_sided) {
    z_alpha <- qnorm(1 - alpha / 2)
  }
  
  E <- 4 * (z_alpha + z_beta)^2 / (log(hr))^2
  return(E)
}

# 示例：HR = 0.75, 80% 效能, 单侧 α = 0.025
E <- calc_schoenfeld(hr = 0.75, alpha = 0.025, power = 0.8)
cat("所需事件数:", ceiling(E), "\n")

# 方法2：使用 survexp（基于人群生存率推算总样本量）
# 假设入组期 24 个月，随访期 24 个月，对照组中位生存 18 个月
lambda_C <- log(2) / 18  # 对照组事件率

# 均匀入组下的事件概率
a <- 24  # 入组月数
f <- 24  # 随访月数
p_event <- 1 - (1 - exp(-lambda_C * f)) / (lambda_C * a) * (1 - exp(-lambda_C * a))

N <- ceiling(E / p_event)
cat("所需总样本量:", N, "\n")
```

#### 4.3.1 非劣效生存试验

```r
# 非劣效生存：HR_NI = 1.3（试验组风险不超过对照组的 1.3 倍）
# 假设真实 HR = 1.0（两组等效）

calc_ni_survival <- function(hr_ni, hr_true = 1, alpha = 0.025, power = 0.8) {
  z_alpha <- qnorm(1 - alpha)
  z_beta <- qnorm(power)
  
  E <- (z_alpha + z_beta)^2 / (log(hr_true) - log(hr_ni))^2
  return(E)
}

E_ni <- calc_ni_survival(hr_ni = 1.3, hr_true = 1.0)
cat("非劣效试验所需事件数:", ceiling(E_ni), "\n")
```

### 4.4 clinfun 包 — 临床试验专用

```r
library(clinfun)

# 4.4.1 非劣效二分类终点（qnbinom 近似）
# 使用 clinfun 的非劣效样本量函数
# 注意：clinfun 主要提供特定设计下的样本量函数

# 示例：非劣效二分类终点
# p0 = 对照组事件率, p1 = 试验组事件率（假设）
# delta = 非劣效界值

sample_size_ni_binom <- function(p0, p1, delta, alpha = 0.025, power = 0.8) {
  # 使用 Fleiss 公式（未连续性校正）
  z_alpha <- qnorm(1 - alpha)
  z_beta <- qnorm(power)
  
  p_bar <- (p0 + p1) / 2
  
  numerator <- (z_alpha * sqrt(2 * p_bar * (1 - p_bar)) + 
                z_beta * sqrt(p0 * (1 - p0) + p1 * (1 - p1)))^2
  
  denominator <- (delta - (p0 - p1))^2
  
  n <- numerator / denominator
  return(ceiling(n))
}

n_ni <- sample_size_ni_binom(p0 = 0.30, p1 = 0.30, delta = 0.10, 
                              alpha = 0.025, power = 0.8)
cat("非劣效二分类试验每组样本量:", n_ni, "\n")

# 4.4.2 等效二分类终点（TOST）
sample_size_equiv_binom <- function(p0, p1, delta, alpha = 0.05, power = 0.8) {
  # 双侧 α = 0.05
  z_alpha <- qnorm(1 - alpha / 2)
  z_beta <- qnorm(power)
  
  p_bar <- (p0 + p1) / 2
  
  numerator <- (z_alpha * sqrt(2 * p_bar * (1 - p_bar)) + 
                z_beta * sqrt(p0 * (1 - p0) + p1 * (1 - p1)))^2
  
  denominator <- (delta - abs(p0 - p1))^2
  
  n <- numerator / denominator
  return(ceiling(n))
}

n_equiv <- sample_size_equiv_binom(p0 = 0.50, p1 = 0.50, delta = 0.15,
                                    alpha = 0.05, power = 0.8)
cat("等效二分类试验每组样本量:", n_equiv, "\n")
```

### 4.7 完整示例：综合样本量计算脚本

```r
#!/usr/bin/env Rscript
# 样本量计算综合示例

library(pwr)
library(survival)
library(rpact)

# ============================================
# 场景1：优效试验 — 连续终点（血压降低）
# ============================================
cat("=== 场景1：优效试验（连续终点）===\n")
result1 <- pwr.t.test(
  d = 5/12,               # 期望降低 5mmHg，SD = 12
  power = 0.9,
  sig.level = 0.025,
  type = "two.sample",
  alternative = "greater"
)
cat(sprintf("每组样本量: %d\n\n", ceiling(result1$n)))

# ============================================
# 场景2：非劣效试验 — 二分类终点（治愈率）
# ============================================
cat("=== 场景2：非劣效试验（二分类终点）===\n")
p0 <- 0.80     # 对照组治愈率
delta_ni <- 0.10  # 非劣效界值 10%
z_alpha <- qnorm(1 - 0.025)
z_beta <- qnorm(0.9)

n_ni <- (z_alpha * sqrt(2 * p0 * (1 - p0)) + 
         z_beta * sqrt(2 * p0 * (1 - p0)))^2 / delta_ni^2
cat(sprintf("每组样本量: %d\n\n", ceiling(n_ni)))

# ============================================
# 场景3：等效试验 — 生存终点
# ============================================
cat("=== 场景3：等效生存试验 ===\n")
hr_ni <- 1.25  # 等效界值 HR = 1.25
hr_true <- 1   # 假设真实 HR = 1

E_equiv <- (qnorm(1 - 0.05/2) + qnorm(0.9))^2 / (log(hr_ni))^2
cat(sprintf("所需事件数: %d\n", ceiling(E_equiv)))

# 推算总样本量（中位生存 12 个月，入组 24 个月，随访 12 个月）
lambda_C <- log(2) / 12
a <- 24; f <- 12
p_event <- 1 - (1 - exp(-lambda_C * f)) / (lambda_C * a) * (1 - exp(-lambda_C * a))
N_equiv <- ceiling(E_equiv / p_event)
cat(sprintf("所需总样本量: %d\n\n", N_equiv))

# ============================================
# 场景4：组序贯设计
# ============================================
cat("=== 场景4：组序贯设计 ===\n")
gs <- gsDesign(k = 3, test.type = 1, alpha = 0.025, beta = 0.1,
               timing = c(0.33, 0.67, 1))
cat("各分析时点边界:\n")
print(gs$upper)
```

## 5. 常见陷阱和注意事项

### 5.1 随机化相关陷阱

| 陷阱 | 后果 | 解决方案 |
|------|------|---------|
| **固定区组长度** | 最后几位受试者分组可被预测 → 选择偏倚 | 使用可变区组长度（4、6、8 随机） |
| **分层过多** | 每层样本量不足，失去均衡意义 | 限制分层因素 ≤ 3 个，每层预期样本量 > 20 |
| **动态随机化无随机成分** | 完全确定性 → 可预测 → 选择偏倚 | 至少 80% 概率分配至最优组，20% 随机 |
| **忽略中心效应** | 多中心试验中心间差异混杂 | 分层随机化时加入"中心"作为分层因素 |
| **随机化序列泄露** | 研究者可能提前知晓分配 → 选择偏倚 | 使用中央随机化系统（IWRS/IVRS） |

---

### 5.2 样本量计算陷阱

#### 5.2.1 参数假设过于乐观

| 问题 | 说明 | 建议 |
|------|------|------|
| 效应量估计过大 | 基于文献或预试验的乐观估计 → 样本量不足 | 使用保守估计，参考历史数据的下限 |
| 标准差估计过小 | 预试验样本量小，SD 估计不准 | 使用文献中报告的最大 SD 或进行敏感性分析 |
| 失访率忽略 | 未考虑脱落 → 实际可评估样本不足 | 预留 10-20% 的额外样本量 |

**敏感性分析示例**：
```r
# 对效应量进行敏感性分析
d_values <- c(0.3, 0.4, 0.5, 0.6)
sapply(d_values, function(d) {
  ceiling(pwr.t.test(d = d, power = 0.8, sig.level = 0.025, 
                      type = "two.sample")$n)
})
# 输出：效应量从 0.6 降到 0.3，样本量从 45 增加到 176
```

#### 5.2.2 非劣效界值设定不当

- **陷阱**：$\delta_{NI}$ 设定过大 → 可能接受无效治疗
- **陷阱**：$\delta_{NI}$ 设定过小 → 样本量过大，试验不可行
- **原则**：
  1. $\delta_{NI}$ 必须小于对照组 vs 安慰剂的历史效应（M1）
  2. 通常取 M1 的 50%（保留至少 50% 疗效）
  3. 需考虑临床意义（即使统计非劣，临床是否可接受？）

#### 5.2.3 多重性未校正

| 场景 | 问题 | 解决方案 |
|------|------|---------|
| 多个主要终点 | I 类错误膨胀 | Bonferroni 校正、Holm 法、Gatekeeping 策略 |
| 多个剂量组 vs 对照组 | 多重比较 | Dunnett 校正 |
| 中期分析 | 累积 I 类错误 | O'Brien-Fleming 或 Pocock 边界 |

#### 5.2.4 生存分析特殊问题

- **事件数 vs 样本量**：生存试验的样本量取决于事件数，而非总人数
- **入组时间估计不准**：实际入组速度慢 → 总随访时间延长 → 事件数不足
- **竞争风险**：非癌症死亡等竞争事件 → 实际事件数减少
- **解决方案**：预留更长的入组期，进行中期事件数评估

---

### 5.3 监管与报告要求

| 要求 | 说明 |
|------|------|
| **预先注册** | 样本量计算方法和参数必须在试验方案中预先定义 |
| **敏感性分析** | 报告不同参数假设下的样本量范围 |
| **中期分析计划** | 如有中期分析，需预先定义停止规则和样本量重估方法 |
| **ITT vs PP** | 样本量计算基于 ITT 人群，但需考虑 PP 人群可能的脱落 |
| **ICH E9 指南** | 遵循 ICH E9(R1) 关于估计目标和样本量计算的原则 |

### 5.4 实用检查清单

```
□ 随机化方法是否与试验规模和复杂度匹配？
□ 是否使用了可变区组长度？
□ 分层因素是否 ≤ 3 个且临床合理？
□ 样本量计算参数是否有文献/预试验支持？
□ 是否进行了敏感性分析（效应量、SD、失访率）？
□ 非劣效界值是否有历史数据支持？
□ 是否考虑了多重性校正？
□ 生存试验是否基于事件数计算？
□ 是否预留了足够的失访/脱落缓冲（通常 10-20%）？
□ 中期分析（如有）是否预先定义了边界和样本量重估规则？
□ 样本量计算是否与统计分析计划一致？
□ 方案中是否完整记录了所有假设和参数来源？
```
