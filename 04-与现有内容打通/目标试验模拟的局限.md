## 1. 目标试验模拟的核心假设

目标试验模拟的有效性依赖于以下关键假设，违反任一假设都可能导致因果效应估计偏倚：

### 1.1 可交换性假设（Exchangeability / No Unmeasured Confounding）

> **假设内容**：在给定测量到的基线协变量条件下，治疗组和对照组的潜在结局分布相同。

$$Y^a \perp A \mid L$$

其中：
- $Y^a$：在治疗方案$a$下的潜在结局
- $A$：实际接受的治疗
- $L$：测量到的基线协变量集合

**实际含义**：所有影响治疗选择和结局的混杂因素都已被测量并纳入调整模型。

**挑战**：
- 观察性研究中，总存在未测量的混杂因素（如疾病严重程度、医生偏好、患者依从性倾向）
- 电子健康记录（EHR）数据尤其缺乏社会行为因素、患者报告结局等变量

### 1.2 一致性假设（Consistency）

> **假设内容**：观察到的结局等于对应实际治疗方案的潜在结局。

$$Y = Y^A$$

**实际含义**：
- 观察到的治疗组患者的结局，等于如果该患者接受该治疗方案时的潜在结局
- 治疗方案定义必须明确且可操作化

**挑战**：
- 观察数据中的"治疗"往往异质性高（剂量、持续时间、依从性不同）
- 难以区分"启动治疗"和"持续治疗"效应

### 1.3 正定性假设（Positivity / Overlap）

> **假设内容**：对于任意协变量组合，每个患者都有接受任一治疗方案的非零概率。

$$0 < P(A=a \mid L) < 1, \quad \forall L$$

**实际含义**：
- 不存在某些患者"只能"接受某一种治疗的情况
- 倾向评分模型中，每个患者的倾向评分不应为0或1

**挑战**：
- 临床实践中存在明确的适应症/禁忌症（如严重肝病患者不能使用某药物）
- 极端倾向评分导致权重不稳定，需截断处理

### 1.4 无干扰假设（No Interference / SUTVA）

> **假设内容**：一个患者的治疗分配不影响其他患者的潜在结局。

**挑战**：
- 群体干预（如疫苗接种）存在间接效应
- 医院层面的政策变化影响所有患者

### 1.5 时间结构假设

> **假设内容**：时间零点（time zero）的定义在观察数据中可准确识别，且随访起点与RCT一致。

**挑战**：
- 观察数据中"治疗启动"时间点可能模糊（处方日期 vs. 实际用药日期）
-  immortal time bias（不朽时间偏倚）是常见违反形式

## 2. 主要局限性

### 2.1 未测量混杂（Unmeasured Confounding）

#### 问题描述
这是TTE最核心的局限。即使调整了大量协变量，仍可能存在影响治疗选择和结局的未测量因素。

#### 常见未测量混杂因素

| 类别 | 具体因素 | 影响方向 |
|------|----------|----------|
| 临床因素 | 疾病严重程度评分、并发症细节、器官功能指标 | 高风险患者更可能接受积极治疗 |
| 行为因素 | 吸烟、饮酒、运动、饮食、服药依从性 | 健康行为影响治疗选择和结局 |
| 社会因素 | 社会经济地位、教育水平、保险类型、社会支持 | 影响治疗可及性和随访 |
| 医疗系统因素 | 医生偏好、医院级别、地区差异、治疗时机 | 影响治疗分配 |
| 患者报告因素 | 症状主观评分、生活质量、治疗偏好 | 难以从EHR获取 |

#### 后果
- 效应估计可能有方向性偏倚（高估或低估）
- 无法通过增加样本量解决
- 倾向评分匹配/加权只能平衡测量到的协变量

#### 缓解策略
1. **使用代理变量**：用可用变量间接反映未测量因素（如用急诊次数代理健康状况）
2. **负对照结局/暴露**：选择理论上不应受治疗影响的结局/暴露检验混杂
3. **工具变量（IV）**：利用外生变异来源（如医生偏好、地理变异）
4. **敏感性分析**：量化未测量混杂对结果的影响（见第3节）

---

### 2.2 选择偏倚（Selection Bias）

#### 2.2.1 入组偏倚（Eligibility Bias）

**问题**：观察数据中符合目标试验入组标准的患者群体，与RCT目标人群不同。

**例子**：
- RCT常排除老年人、合并症患者、孕妇
- EHR数据中这些人群占很大比例，但TTE模拟时仍需应用相同排除标准
- 结果外推性受限

**缓解**：
- 明确报告目标试验与模拟人群的差异
- 使用外部验证队列评估外推性

#### 2.2.2 存活偏倚（Survivor Bias / Prevalent User Bias）

**问题**：观察数据中纳入的是"存活到观察期开始"的患者，排除了早期死亡/失访者。

**例子**：
- 研究药物长期安全性时，纳入的是已存活使用药物一段时间的患者
- 早期严重不良反应患者已被排除

**缓解**：
- 采用新使用者设计（new user design）
- 明确时间零点，从治疗启动开始随访

#### 2.2.3 脱落/失访偏倚（Attrition Bias）

**问题**：随访期间患者失访与结局相关，导致结局分布偏倚。

**例子**：
- 病情恶化患者失访（结局更差但未被记录）
- 病情好转患者失访（结局更好但未被记录）

**缓解**：
- 使用逆概率删失加权（IPCW）
- 多重插补处理缺失结局数据

#### 2.2.4 时间相关偏倚（Time-Related Biases）

| 偏倚类型 | 描述 | 示例 |
|----------|------|------|
| 不朽时间偏倚 | 治疗组患者在"等待治疗"期间被错误归入治疗组，这段时间内不可能发生结局 | 研究药物预防死亡，但患者需存活到用药才能入组治疗组 |
| 时间窗偏倚 | 暴露和结局的观察时间窗定义不一致 | 用药后30天内结局归因于药物，但未用药组观察期更长 |
| 领先时间偏倚 | 早期诊断导致生存时间"虚增" | 筛查项目早期发现疾病，生存期延长但实际死亡时间未变 |

**缓解**：
- 严格定义时间零点
- 使用克隆-审查-加权法（clone-censor-weight）
- 时间依赖协变量分析

---

### 2.3 信息偏倚（Information Bias）

#### 2.3.1 暴露测量误差

**问题**：观察数据中的治疗暴露定义不精确。

**来源**：
- 处方记录 ≠ 实际用药（依从性问题）
- 用药剂量/持续时间记录不完整
- 治疗切换/交叉未被记录

**后果**：
- 非差分误差：倾向于低估效应（偏向无效）
- 差分误差：可能高估或低估效应

**缓解**：
- 使用药房填充数据验证用药
- 定义"持续用药"标准（如连续填充）
- 敏感性分析：假设不同依从性水平

#### 2.3.2 结局测量误差

**问题**：结局的识别和分类不准确。

**来源**：
- ICD编码敏感性/特异性问题
- 结局定义与RCT不一致
- 结局评估者未盲法

**例子**：
- 使用ICD代码识别心肌梗死，但代码阳性预测值仅70-80%
- 死亡原因编码错误

**缓解**：
- 使用验证子集估计测量误差参数
- 多重插补校正结局误分类
- 使用更严格的结局定义（如结合实验室+诊断+用药）

#### 2.3.3 协变量测量误差

**问题**：基线协变量测量不精确，影响混杂控制效果。

**来源**：
- 实验室值单次测量 vs. 长期平均水平
- 自我报告变量的回忆偏倚
- 时间变化的协变量仅用基线值代表

**缓解**：
- 使用多次测量平均值
- 时间依赖协变量模型

## 3. 敏感性分析方法

敏感性分析用于评估研究结果对核心假设违反的稳健性。以下是TTE常用的敏感性分析方法：

### 3.1 未测量混杂的敏感性分析

#### 3.1.1 E值（E-value）

**概念**：使观察到的关联被解释为完全由未测量混杂导致所需的最小混杂强度。

**计算公式**（针对RR）：

$$E\text{-value} = RR + \sqrt{RR \times (RR - 1)}$$

**解读**：
- E值越大，结果对未测量混杂越稳健
- 若E值大于已知混杂因素的典型强度，则结果较可信

**R实现**：
```r
# 使用EValue包
library(EValue)

# 计算E值
evalue(hr = 0.75, lo = 0.60, hi = 0.94)

# 输出示例：
# E-value: 1.71
# CI limit closest to null: 0.60
# E-value for CI limit: 1.42
```

**局限性**：
- 假设未测量混杂与暴露和结局均二分类
- 未考虑混杂因素间的相互作用

#### 3.1.2 偏倚函数法（Bias Function Approach）

**概念**：指定未测量混杂与暴露、结局的关联强度，重新计算校正后的效应估计。

**参数**：
- $RR_{UY}$：未测量混杂与结局的关联强度
- $RR_{AU}$：未测量混杂与暴露的关联强度
- $P(U=1|A=1)$：治疗组中未测量混杂的暴露率

**R实现**：
```r
# 使用biasAnalysis包（概念性代码）
library(biasAnalysis)

# 指定参数
bias_correct <- biasAdjust(
  obsRR = 0.75,
  RRU = c(1.5, 2.0, 3.0),  # 未测量混杂-结局关联
  RRA = c(1.5, 2.0, 3.0),  # 未测量混杂-暴露关联
  pu1 = 0.3,  # 治疗组中U的暴露率
  pu0 = 0.2   # 对照组中U的暴露率
)
```

#### 3.1.3 外部信息校正法

**概念**：利用外部数据源估计未测量混杂的分布和关联强度，进行定量偏倚分析。

**步骤**：
1. 从外部研究获取未测量混杂的患病率和关联强度
2. 使用偏倚公式校正效应估计
3. 进行情景分析（最好/最坏情景）

---

### 3.2 选择偏倚的敏感性分析

#### 3.2.1 逆概率加权敏感性分析

**概念**：对不同的失访/审查机制假设，计算不同权重下的效应估计。

**参数**：
- 假设失访患者与存活患者的结局风险比

**情景**：
| 情景 | 失访患者风险比 | 校正HR |
|------|----------------|--------|
| 基准 | 1.0（与存活者相同） | 0.75 |
| 温和 | 1.5（失访者风险更高） | 0.78 |
| 极端 | 3.0（失访者风险极高） | 0.85 |

#### 3.2.2 模式混合模型（Pattern Mixture Model）

**概念**：按失访模式分层，假设不同模式下结局分布不同。

**应用**：
- 适用于纵向结局数据
- 可结合多重插补

---

### 3.3 信息偏倚的敏感性分析

#### 3.3.1 结局误分类校正

**概念**：假设结局测量的敏感性（Se）和特异性（Sp），校正效应估计。

**校正公式**（针对RR）：

$$RR_{corrected} = \frac{RR_{observed} \times (Se + Sp - 1)}{Se - (1-Sp) \times RR_{observed}}$$

**R实现**：
```r
# 结局误分类校正函数
correct_misclassification <- function(RR_obs, Se, Sp) {
  RR_corr <- (RR_obs * (Se + Sp - 1)) / (Se - (1 - Sp) * RR_obs)
  return(RR_corr)
}

# 情景分析
Se_range <- c(0.7, 0.8, 0.9, 1.0)
Sp_range <- c(0.9, 0.95, 0.98, 1.0)

# 生成校正结果矩阵
```

#### 3.3.2 暴露误分类校正

**概念**：假设用药依从性水平，校正效应估计。

**情景**：
| 依从性假设 | 暴露错误分类率 | 校正HR |
|------------|----------------|--------|
| 完全依从 | 0% | 0.70 |
| 80%依从 | 20%非差分错误 | 0.73 |
| 60%依从 | 40%非差分错误 | 0.78 |

---

### 3.4 综合敏感性分析框架

#### 3.4.1 情景分析（Scenario Analysis）

**方法**：设定一系列合理的情景，报告每种情景下的效应估计。

**示例表格**：

| 情景 | 假设 | HR (95% CI) | 结论 |
|------|------|-------------|------|
| 主分析 | 无未测量混杂 | 0.75 (0.60-0.94) | 显著 |
| 情景1 | E值=1.5的混杂存在 | 0.79 (0.63-0.99) | 边缘显著 |
| 情景2 | E值=2.0的混杂存在 | 0.85 (0.67-1.07) | 不显著 |
| 情景3 | 结局Se=0.8, Sp=0.95 | 0.77 (0.62-0.96) | 显著 |
| 情景4 | 失访者风险比=2.0 | 0.80 (0.64-1.00) | 边缘显著 |

#### 3.4.2 定量偏倚分析（Quantitative Bias Analysis, QBA）

**工具**：
- R包：`EValue`, `biasAnalysis`, `episensr`
- Stata包：`evalue`, `bias`

**流程**：
1. 识别潜在偏倚来源
2. 指定偏倚参数分布（点估计或范围）
3. 蒙特卡洛模拟生成校正后的效应分布
4. 报告校正后的点估计和不确定性

```r
# 使用episensr进行QBA示例
library(episensr)

# 未测量混杂校正
mcnaive(
  data = mydata,
  exposure = "treatment",
  outcome = "event",
  confounder = "unmeasured_proxy",
  iter = 1000
)
```

## 4. 何时不适合用目标试验模拟

### 4.1 数据条件不满足

| 情况 | 原因 | 替代方案 |
|------|------|----------|
| 关键混杂因素无法测量 | 如疾病严重程度、患者偏好 | 考虑RCT或准实验设计（如RD、IV） |
| 样本量不足 | 稀有暴露/结局，事件数少 | 多中心数据整合或放弃 |
| 随访时间不足 | 长期结局无法观察 | 使用替代结局或放弃 |
| 数据质量差 | 暴露/结局测量误差大 | 数据收集改进或放弃 |

### 4.2 研究问题不匹配

| 情况 | 原因 | 替代方案 |
|------|------|----------|
| 研究治疗效果异质性 | TTE估计平均效应，个体效应需更多假设 | 亚组分析+谨慎解释 |
| 研究剂量-反应关系 | 观察数据中剂量选择高度混杂 | RCT或 Mendelian Randomization |
| 研究罕见不良事件 | 需要极大样本量 | 药物警戒数据库、自发报告系统 |
| 研究短期效应但数据只有长期 | 时间零点定义困难 | 重新设计或放弃 |

### 4.3 存在更优设计

| 情况 | 原因 | 替代方案 |
|------|------|----------|
| 可开展RCT | RCT是金标准 | 优先RCT |
| 存在自然实验 | 如政策变化、地理变异 | 断点回归、差异中的差异 |
| 有强工具变量 | 如医生偏好、距离 | 工具变量分析 |

### 4.4 伦理/可行性限制

- **伦理问题**：某些暴露无法随机化（如吸烟、环境污染），但TTE仍可使用
- **可行性问题**：目标试验模拟需要高质量、详细的观察数据，若数据不可得则不适合

### 4.5 决策建议流程图

```
开始
  │
  ▼
能否开展RCT？───是──→ 优先RCT
  │否
  ▼
观察数据是否满足条件？
  │
  ├─ 关键混杂可测量？───否──→ 考虑IV/负对照/放弃
  │  │是
  │  ▼
  ├─ 样本量充足？───否──→ 多中心整合/放弃
  │  │是
  │  ▼
  ├─ 暴露/结局定义清晰？───否──→ 数据改进/放弃
  │  │是
  │  ▼
  └─ 时间结构可识别？───否──→ 重新定义时间零点/放弃
     │是
     ▼
  进行TTE + 敏感性分析
```

## 6. 总结与关键要点

### 6.1 核心假设检查清单

在进行TTE时，逐一检查以下假设：

- [ ] **可交换性**：是否已测量所有重要混杂因素？是否有负对照检验？
- [ ] **一致性**：治疗方案定义是否明确？观察到的治疗是否等于目标治疗？
- [ ] **正定性**：倾向评分是否有极端值？是否有患者只能接受一种治疗？
- [ ] **无干扰**：患者之间是否存在相互影响？
- [ ] **时间结构**：时间零点是否清晰？是否存在不朽时间偏倚？

### 6.2 敏感性分析必做项目

| 分析类型 | 方法 | 报告内容 |
|----------|------|----------|
| 未测量混杂 | E值计算 | E值及对应混杂强度解释 |
| 选择偏倚 | IPCW敏感性 | 不同失访假设下的效应范围 |
| 信息偏倚 | 误分类校正 | 不同Se/Sp假设下的校正结果 |
| 综合 | 情景分析 | 主分析+情景分析的对比表 |

### 6.3 报告建议

遵循TARGET指南（2025 JAMA），特别强调：

1. **明确报告目标试验与模拟的差异**（条目19）
2. **讨论局限性**（条目21）
3. **报告敏感性分析结果**（条目18）
