## 1. 非劣效试验设计原理

### 1.1 核心概念

**非劣效试验（Non-inferiority Trial）** 的目标是证明新疗法**不差于**现有标准疗法（超过一个预设的界值），而非证明其更优。

| 试验类型 | 研究假设 | 临床目的 |
|---------|---------|---------|
| **优效试验** | 新疗法 > 对照 | 证明新疗法更好 |
| **非劣效试验** | 新疗法 ≥ 对照 - δ | 证明新疗法"足够好" |
| **等效试验** | \|新疗法 - 对照\| ≤ δ | 证明两者效果相当 |

### 1.2 设计逻辑

```
                    无效应差值 (0)
                          │
    非劣效界值 (−δ) ──────┼─────────────────→ 优效方向
                          │
    ┌─────────────────────┴─────────────────────┐
    │                                           │
    │   非劣效区域          优效区域             │
    │   (−δ, 0]            (0, +∞)              │
    │                                           │
    └───────────────────────────────────────────┘
```

**关键假设**：
- 对照疗法相对于安慰剂有已知的、稳定的效应（**活性对照效应**）
- 新疗法与对照疗法的效应差异在界值内可接受
- 试验质量（依从性、盲法、失访率）足够高，避免**效应稀释（assay sensitivity）损失**

### 1.3 适用场景

| 场景 | 说明 | 示例 |
|------|------|------|
| 简化给药方案 | 新疗法更方便但预期效果略低 | 口服药 vs 静脉注射 |
| 降低副作用 | 新疗法安全性更好 | 低剂量 vs 标准剂量 |
| 降低成本 | 仿制药或生物类似药 | 生物类似药 vs 原研药 |
| 特殊人群 | 标准疗法不适用时 | 肾功能不全患者的替代方案 |
| 伦理考量 | 无法使用安慰剂对照 | 已有有效疗法时 |

## 2. 非劣效界值选择（M1、M2、delta）

### 2.1 界值定义框架

```
┌─────────────────────────────────────────────────────────────┐
│                    界值选择逻辑链                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  M1 = 对照 vs 安慰剂的效应（历史证据）                        │
│       ↓                                                     │
│  M2 = 临床可接受的最大损失（专家共识）                        │
│       ↓                                                     │
│  δ  = min(M1, M2) 或 M2（保守原则）                         │
│                                                             │
│  原则：保留对照效应的一部分（通常50%）                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 参数详解

| 参数 | 定义 | 来源 |
|------|------|------|
| **M1** | 对照疗法相对于安慰剂的效应量（历史Meta分析） | 历史RCTs的Meta分析 |
| **M2** | 临床可接受的最大效应损失 | 临床专家共识 + 监管指南 |
| **δ** | 非劣效界值 | M2（保守）或 M1 × 保留比例 |

### 2.3 界值选择的"50%原则"

**FDA指南推荐**：非劣效界值不应超过历史对照效应的50%。

```
保留效应 = M1 − δ ≥ 50% × M1
即：δ ≤ 50% × M1
```

**示例计算**：
- 历史Meta分析显示：对照药 vs 安慰剂的HR = 0.70（30%风险降低）
- M1 = 1 − 0.70 = 0.30（绝对效应差）
- 50%原则：δ ≤ 0.30 × 50% = 0.15
- 非劣效界值：HR上限 ≤ 1.15（或绝对差 ≤ 15%）

### 2.4 界值类型

| 类型 | 适用终点 | 表示方式 |
|------|---------|---------|
| **绝对差** | 连续变量、率差 | δ = 5%、10 mmHg |
| **相对比** | 二分类、生存分析 | HR ≤ 1.25、RR ≤ 1.30 |
| **风险比** | 时间-事件终点 | 95% CI上限 < 界值 |

### 2.5 界值选择的挑战

| 挑战 | 说明 | 应对策略 |
|------|------|---------|
| 历史对照效应不稳定 | 不同时代、人群效应不同 | 敏感性分析、保守估计 |
| 间接比较偏差 | 历史试验与当前试验不可比 | 严格纳入标准、调整分析 |
| 临床意义 vs 统计意义 | 小界值导致样本量过大 | 平衡临床合理性与可行性 |
| 监管差异 | FDA、EMA界值标准不同 | 提前沟通、参考ICH E9(R1) |

## 3. 比较效果研究（CER）方法

### 3.1 定义与范围

**比较效果研究（Comparative Effectiveness Research, CER）**：
> 评估不同干预措施在真实世界中的相对效果，为临床决策和卫生政策提供证据。

**核心特征**：
- 比较**多个实际可用的选项**（而非安慰剂）
- 关注**患者有意义的结局**（而非替代终点）
- 纳入**多样化患者群体**（而非严格筛选）
- 反映**真实临床实践**（而非理想化条件）

### 3.2 CER vs RCT

| 维度 | RCT（优效/非劣效） | CER（真实世界） |
|------|-------------------|----------------|
| **设计** | 随机化、对照 | 观察性队列、目标试验模拟 |
| **人群** | 严格纳入排除 | 广泛、代表性 |
| **干预** | 标准化方案 | 真实用药模式 |
| **依从性** | 高（监控） | 真实水平 |
| **偏倚控制** | 随机化 | 倾向评分、因果推断 |
| **外部效度** | 有限 | 高 |
| **内部效度** | 高 | 需严格方法学控制 |

### 3.3 CER核心方法

#### 3.3.1 目标试验模拟（Target Trial Emulation）

```
┌─────────────────────────────────────────────────────────────┐
│              目标试验模拟框架                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 定义目标试验（理想RCT）                                  │
│     ├── 纳入/排除标准                                        │
│     ├── 治疗策略（新用药设计）                                │
│     ├── 结局定义                                             │
│     ├── 随访时间                                             │
│     └── 分析集（ITT vs per-protocol）                       │
│                                                             │
│  2. 从观察性数据模拟                                         │
│     ├── 新用药队列（避免 prevalent user bias）              │
│     ├── 基线时间对齐（t=0）                                  │
│     ├── 排除条件（模拟随机化）                                │
│     └── 随访起始一致                                         │
│                                                             │
│  3. 偏倚控制                                                 │
│     ├── 倾向评分匹配/加权                                    │
│     ├── 时间依赖性混杂（time-varying confounding）          │
│     └── 未测量混杂敏感性分析                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 3.3.2 倾向评分方法

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **匹配（Matching）** | 1:1或1:k匹配，平衡协变量 | 样本充足、重叠好 |
| **加权（IPTW）** | 逆概率加权，创建伪人群 | 保留样本量 |
| **分层（Stratification）** | 按PS分 quintile 分析 | 简单、直观 |
| **协变量调整** | PS作为协变量纳入模型 | 样本小、重叠差 |

#### 3.3.3 因果推断方法

| 方法 | 适用场景 | 优势 |
|------|---------|------|
| **G-公式（G-computation）** | 时间依赖性混杂 | 理论最优 |
| **IPTW + MSME** | 纵向数据 | 处理时变混杂 |
| **工具变量（IV）** | 未测量混杂 | 解决遗漏变量 |
| **断点回归（RDD）** | 存在阈值规则 | 局部因果效应 |

### 3.4 CER报告规范

| 规范 | 适用 | 核心要求 |
|------|------|---------|
| **STROBE** | 观察性研究 | 6项核心条目 + 观察性研究 checklist |
| **RECORD** | 使用常规数据 | STROBE + 数据质量、链接说明 |
| **ISPE Good Practice** | CER最佳实践 | 目标试验框架、偏倚评估 |

## 4. 非劣效与CER的关联

### 4.1 概念交叉

```
                    ┌─────────────────────────────────┐
                    │      比较效果研究（CER）         │
                    │  （比较实际可用的干预措施）       │
                    └───────────────┬─────────────────┘
                                    │
          ┌─────────────────────────┼─────────────────────────┐
          │                         │                         │
          ▼                         ▼                         ▼
   ┌──────────────┐        ┌──────────────┐        ┌──────────────┐
   │  优效比较     │        │ 非劣效比较    │        │  等效比较    │
   │  (Superiority)│        │(Non-inferiority)│      │ (Equivalence)│
   └──────────────┘        └──────────────┘        └──────────────┘
          │                         │                         │
          │    RCT或观察性研究均可    │    通常需RCT设计       │
          │                         │                         │
          └─────────────────────────┴─────────────────────────┘
                                    │
                    ┌───────────────▼─────────────────┐
                    │   CER中的非劣效：真实世界非劣效  │
                    │   （观察性数据 + 非劣效框架）     │
                    └─────────────────────────────────┘
```

### 4.2 CER中的非劣效应用

**场景**：真实世界中比较两种实际可用的疗法，新疗法有便利/安全/成本优势，预期效果略低但可接受。

| 维度 | RCT非劣效 | CER非劣效 |
|------|----------|----------|
| **界值来源** | 历史RCT Meta分析 | 历史证据 + 真实世界数据 |
| **偏倚控制** | 随机化 | 倾向评分 + 因果推断 |
| **效应稀释风险** | 低（严格设计） | 高（需敏感性分析） |
| **结论强度** | 高 | 中等（需谨慎解释） |

### 4.3 CER非劣效的特殊挑战

| 挑战 | 说明 | 解决方案 |
|------|------|---------|
| **效应稀释（Assay Sensitivity）** | 观察性数据无法证明试验"能检测效应" | 阳性对照分析、历史数据校准 |
| **未测量混杂** | 倾向评分无法平衡未测量变量 | 工具变量、E-value敏感性分析 |
| **界值外推** | 历史RCT界值可能不适用于真实人群 | 真实世界界值校准、敏感性分析 |
| **依从性差异** | 真实世界依从性低于RCT | per-protocol分析、ITT+PP双重报告 |

### 4.4 分析策略建议

```r
# CER非劣效分析框架（伪代码）
# 1. 目标试验模拟定义
target_trial <- list(
  eligibility = c("new users", "no prior exposure"),
  treatment = c("Drug A", "Drug B"),
  outcome = "MACE",
  follow_up = "3 years"
)

# 2. 倾向评分匹配
matched_cohort <- matchit(treatment ~ covariates, data = observational_data)

# 3. 非劣效检验
cox_model <- coxph(Surv(time, event) ~ treatment, data = matched_data)
hr_ci <- confint(cox_model)

# 4. 非劣效判断
if (hr_ci[2] < delta_noninferiority) {
  conclusion <- "Non-inferior"
} else {
  conclusion <- "Non-inferiority not established"
}

# 5. 敏感性分析
# - 不同PS模型 specification
# - 不同匹配方法（1:1, 1:k, IPTW）
# - E-value计算（未测量混杂）
# - 阳性对照效应验证
```

### 5.4 从CER到非劣效的转换

如果 diabetes-study 的结果是 HR = 1.10 (95% CI: 0.95-1.28)，如何判断非劣效？

```
步骤1：确定界值 δ
  - 历史证据：Sitagliptin vs 安慰剂的效应（需文献检索）
  - 假设 M1 = 15%风险降低（HR = 0.85）
  - 50%原则：δ ≤ 0.15 × 50% = 0.075
  - 非劣效界值：HR上限 ≤ 1.08

步骤2：非劣效检验
  - 观察到的HR = 1.10，95% CI上限 = 1.28
  - 1.28 > 1.08 → 非劣效不成立

步骤3：敏感性分析
  - 不同δ值（10%、15%、20%）
  - 不同PS模型
  - E-value计算
```

## 6. 关键要点总结

| 主题 | 核心要点 |
|------|---------|
| **非劣效设计** | 证明"不差于"而非"优于"，需预设界值δ |
| **界值选择** | M1（历史效应）→ M2（临床可接受）→ δ = min(M1, M2)，50%原则 |
| **CER方法** | 目标试验模拟 + 倾向评分 + 因果推断，STROBE/RECORD报告 |
| **关联** | CER可包含非劣效比较，但需处理效应稀释和未测量混杂 |
| **实践** | diabetes-study是CER优效研究，可扩展为非劣效框架 |
