## 二、阶段一：数据源判断（能不能做？）

**核心问题**：我手里的数据能不能回答一个明确的预测问题？

### 2.1 病案首页数据

**5个条件**（[[病案首页真实世界研究可行性判断]]）：

| 条件 | 说明 |
|------|------|
| 诊断编码完整 | ICD-10编码覆盖主要诊断 |
| 变量可提取 | 关键变量在病案首页有对应字段 |
| 随访可追踪 | 能追踪到结局发生 |
| 数据质量可接受 | 缺失率、错误率在可接受范围 |
| 伦理可合规 | 符合伦理审查要求 |

### 2.2 小样本判断

**4步判断框架**（[[小样本研究_100例判断框架]]）：

```
样本量 → 事件数 → 变量数 → 统计方法
```

- 样本量<100：谨慎评估
- 事件数<10：不建议做预测模型
- 变量数/事件数比>1:10：过拟合风险高

### 2.3 多中心数据管理

**5步校准**（[[多中心数据管理_变量字典先行]]）：

```
变量字典 → 缺失处理 → 单位统一 → 时间窗对齐 → 数据质量检查
```

## 三、阶段二：问题定义（做什么？）

**核心问题**：把真实临床困惑转化为一个可测量、可复核、可验证、可解释的问题。

### 3.1 课题选题

**5步路径**（[[临床课题选题_病例多不等于课题成熟]]）：

```
临床困惑 → 文献空白 → 数据可得性 → 方法可行性 → 临床价值
```

### 3.2 临床争议转化为论文切口

**6步转译**（[[临床争议转化为论文切口]]）：

```
临床争议 → 分歧点 → 研究问题 → 假设 → 结局 → 方法
```

### 3.3 预测问题定义

**4个要素**（[[临床预测模型全流程_合集导读]]）：

| 要素 | 核心问题 |
|------|---------|
| 研究对象 | 谁进入模型？连续病例还是特定亚群 |
| 预测时点 | 入院时、术前、治疗前？防止使用未来信息 |
| 候选变量 | 预测时点前可获得、稳定、可复核、可解释 |
| 临床结局 | 定义、判定与时间窗必须具体且可测量 |

## 四、阶段三：研究设计（怎么做？）

**核心问题**：研究设计是否支持因果推断或预测目标？

### 4.1 纳排标准

**6步写作路径**（[[纳排标准_定义研究对象]]）：

```
目标人群 → 纳入标准 → 排除标准 → 时间窗 → 样本量 → 伦理
```

### 4.2 时间窗口设计

**5步时间窗**（[[时间窗口设计_统计结果失真]]）：

```
时间零点 → 暴露窗 → 结局窗 → 删失处理 → 竞争风险
```

### 4.3 统计分析计划（SAP）

**6件事**（[[统计分析计划SAP_跑统计前定6件事]]）：

```
主分析 → 亚组分析 → 敏感性分析 → 缺失处理 → 多重比较 → 软件版本
```

## 六、阶段五：统计分析（怎么算？）

**核心问题**：统计方法是否匹配研究问题？

### 6.1 描述统计

**正态分布 vs 偏态**（[[正态分布与偏态资料_描述统计逻辑]]）：

| 分布 | 描述统计 |
|------|---------|
| 正态分布 | 均数 ± 标准差 |
| 偏态分布 | 中位数（四分位数间距） |

### 6.2 组间比较

**多组比较**（[[三组及以上数据比较_不要先做三次t检验]]）：

```
总体检验（ANOVA/Kruskal-Wallis）→ 事后比较（Tukey/Dunn）
```

### 6.3 P值理解

**6步判断**（[[P值正确理解_统计入门]]）：

```
零假设 → 检验统计量 → P值 → 显著性水平 → 效应量 → 临床意义
```

### 6.4 效应量

**四步阅读法**（[[效应量_统计显著不等于临床重要]]）：

```
统计显著 → 效应量大小 → 置信区间 → 临床意义
```

### 6.5 二分类结局

**四个效应量**（[[RR_OR_ARR_NNT_二分类结局效应量]]）：

| 指标 | 回答的问题 |
|------|-----------|
| RR | 风险按比例变了多少 |
| OR | odds之比（结局罕见时近似RR） |
| ARR | 每100人实际少发生多少事件 |
| NNT | 需治疗多少人避免1个结局 |

### 6.6 生存分析

**KM曲线 + HR + Cox回归**（[[生存分析_KM曲线_HR_Cox回归]]）：

```
KM曲线（时间维度）→ HR（相对风险）→ Cox回归（协变量调整）
```

### 6.7 预测模型

**四维度评价**（[[临床预测模型_高AUC不等于好模型]]）：

| 维度 | 指标 |
|------|------|
| 区分度 | AUC/ROC |
| 校准度 | 校准曲线 |
| 临床净获益 | DCA |
| 外部验证 | 独立队列 |

## 七、阶段六：论文写作（怎么报？）

**核心问题**：结果是否经得起追问？

### 7.1 投稿前自查

**三轮检查**（[[投稿前自查_STROBE_RECORD_TRIPOD]]）：

```
第一轮：STROBE清单（观察性研究）
第二轮：RECORD清单（真实世界数据）
第三轮：TRIPOD清单（预测模型）
```

### 7.2 报告规范

**核心原则**：让读者先看到"实际风险"，再讨论相对效应和统计显著性。

```
数据源判断
    ↓
  病案首页可行性判断（5条件）

    ↓

    ↓
  纳排标准（6步）

  SAP（6件事）
    ↓
数据治理
    ↓
  变量字典（5步校准）

  混杂控制（因果路径）
    ↓
统计分析
    ↓
  描述统计（正态vs偏态）

    ↓
论文写作
    ↓
  投稿前自查（STROBE/RECORD/TRIPOD）
```
