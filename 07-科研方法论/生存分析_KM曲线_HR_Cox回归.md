# 生存分析：KM曲线、HR与Cox回归怎么看？

> 来源：吃透这套统计概念，论文审稿不卡壳⑯
> 提取日期：2026-06-29
> 核心目标：生存分析不是只看死亡人数，而是看"何时、谁、为何发生"
> 关联笔记：[[效应量_统计显著不等于临床重要]]、[[P值正确理解_统计入门]]

---

## 一、核心问题

**生存分析并不只研究"生或死"。它处理的是"从一个明确的时间零点开始，到某个事件发生为止经历了多久"。**

事件可以是死亡、复发、ICU转入、机械通气、再次住院，也可以是达到某个复合临床结局。

**读生存分析论文，关键不是背公式，而是始终追问四个问题**：
1. 谁被纳入？
2. 从何时开始计时？
3. 何事算事件？
4. 哪些人为何没有观察到事件？

---

## 二、先给结论

| 指标 | 回答的问题 |
|------|-----------|
| **KM曲线** | 随时间推移，未发生事件的估计概率如何变化 |
| **HR** | 同一时点的相对瞬时事件风险如何不同 |
| **Cox回归** | 在指定模型假设下，对协变量调整后的HR估计 |

→ 三者不能互相替代。

---

## 三、为什么只看死亡人数，常常会把结论看错？

**案例**：两组各纳入120例患者，12个月内死亡人数相近。若A组多数死亡发生在治疗后1–3个月，而B组的死亡主要在第8个月之后，临床上显然不能说"预后一样"。

**生存资料的核心难点**：每个患者的观察时长通常不同。有的人在第2个月发生事件；有的人到研究结束仍未发生；还有的人中途失访、转院或撤回。最后两类不是"没有事件"，而是我们只知道他们在某一时点之前未发生事件，之后的状态未知。这个现象称为**删失（censoring）**。

> 一个重要前提是删失大致满足非信息性删失：在已纳入模型的信息条件下，被删失者后续发生事件的风险不应系统性地不同于仍在随访者。

---

## 四、KM曲线：读图必看三处

KM方法把随访时间切成一个个事件时点：每发生一次事件，曲线便向下走一个台阶；在两个事件时点之间，曲线保持水平。纵轴通常是"未发生事件的估计概率"，横轴是随访时间。

| 看什么 | 说明 |
|--------|------|
| **01 看曲线何时开始分离** | 如果两条曲线早期迅速分开，提示差异可能主要发生在早期；若后期才分开，则需要结合治疗起效时间、病程阶段和随访机制解释 |
| **02 看删失标记与尾部曲线** | 曲线末端常常只有少数患者仍在险，估计不稳定。若图中没有在险人数，或末端大量删失却仍强行比较"3年生存率"，可解释性会明显下降 |
| **03 看曲线是否交叉** | 交叉意味着两组的相对风险可能随时间变化，单个HR的概括能力会减弱。此时只报一个log-rank P值或一个Cox HR，往往不够 |

---

## 五、HR：不要把它说成"死亡概率减少了X%"

Cox比例风险模型通常写作 h(t|X)=h₀(t)exp(βX)，其中h(t)是时点t附近的瞬时事件风险，HR=exp(β)。

**HR=0.70更严谨的表述**：在模型设定和比例风险假设大致成立的前提下，A组在任一给定时点的瞬时事件风险约为B组的70%。

> HR比较的是同一时点的瞬时事件风险，而非累计死亡概率。这与"死亡概率减少30%"不是同一句话。

### 报告HR时，必须同时看这4件事

| 事项 | 说明 |
|------|------|
| **01 HR的方向** | HR<1：相对瞬时风险更低；HR>1：更高 |
| **02 95% CI** | CI是否跨1，决定估计精确度与统计不确定性 |
| **03 绝对效应** | 补充固定时间点生存率、RMST或绝对风险差 |
| **04 曲线形态** | 明显交叉或分离随时间改变，应怀疑非比例风险 |

---

## 六、Cox回归：不是跑出一张表，就能写"独立危险因素"

Cox回归的价值在于可以同时纳入多项协变量，对已测量的混杂因素进行调整。但"调整后HR"不等于因果效应，更不等于变量本身被证明为独立致病因素。

### 从时间零点到比例风险检验，每一步都会影响HR的临床解释

| 步骤 | 核心任务 |
|------|---------|
| **01 确定时间零点** | 入组、确诊、手术或治疗起始？必须预先定义 |
| **02 定义事件与删失** | 事件判定标准、末次随访、失访与竞争事件如何处理？ |
| **03 明确协变量策略** | 基于临床知识与因果路径，而非只靠单因素P值筛变量 |
| **04 检验比例风险** | Schoenfeld残差、曲线形态或时间交互项；必要时分层或时变效应 |
| **05 完整报告结果** | HR、95%CI、P值、事件数、随访描述、KM曲线与在险人数 |

### 表述规范

把"独立危险因素"换成更准确的表述：在多变量Cox模型中，变量X与更高的事件风险相关（调整后HR…，95%CI…）。

## 七、案例：不死时间偏倚

**场景**：某中心回顾性纳入240例接受手术的肺癌患者，拟比较"围术期强化管理"与"常规管理"对12个月全因死亡的关联。

**错误做法**：把"强化管理完成日"作为分组起点，而常规组从手术日开始计时 → 强化组患者必须先存活到完成管理日才有资格进入暴露组，形成典型的不死时间偏倚。

**更稳妥的做法**：预先定义统一的时间零点（如手术日或符合纳入标准日）；明确强化管理是基线暴露、时变暴露，还是需要采用目标试验模拟框架处理。

---

## 八、比例风险假设不成立时，单个HR还能不能用？

比例风险假设的直观含义是：两组风险的比值在随访期间近似稳定。若KM曲线明显交叉、早期风险上升而后期下降，或者Schoenfeld残差检验提示效应随时间改变，那么一个"全程HR"可能掩盖方向相反的阶段性效应。

**可考虑**：报告分段HR、加入时间交互项、使用时变系数模型，或补充RMST。

---

## 九、读生存分析论文快速核对清单

| 核对点 | 要问的问题 |
|--------|-----------|
| **研究对象** | 纳入与排除标准清楚吗？每组基线人数、事件数和失访情况是否完整？ |
| **时间零点** | 从诊断、入院、手术、治疗开始还是随机化开始计时？两组是否一致？ |
| **结局与删失** | 事件如何判定？末次随访、失访、转院和竞争事件怎样处理？ |
| **KM曲线** | 有无删失标记、在险人数、足够的横轴范围？曲线是否交叉或尾部不稳定？ |
| **Cox结果** | HR、95%CI、P值、协变量选择原则与比例风险检验是否报告？ |
| **临床意义** | 是否同时提供绝对效应、随访时长和不确定性，而不是只强调"HR<1"？ |

---

## 十、写在最后

> 生存分析的难点不在于画出一张曲线，而在于让时间、删失、研究设计和模型假设彼此一致。

**真正可靠的阅读顺序**：
1. 先看研究设计和时间零点
2. 再看曲线与在险人数
3. 随后看HR及其CI
4. 最后核对模型假设与调整逻辑

## 十二、关键数字速查

| 指标 | 说明 |
|------|------|
| 核心问题 | 从时间零点到事件发生经历了多久 |
| 四个追问 | 谁被纳入？+ 从何时开始计时？+ 何事算事件？+ 哪些人为何没有观察到事件？ |
| KM曲线读图 | 看分离时间 + 看删失与尾部 + 看是否交叉 |
| HR报告 | 方向 + 95%CI + 绝对效应 + 曲线形态 |
| Cox回归5步 | 时间零点 → 事件与删失 → 协变量策略 → 比例风险检验 → 完整报告 |
