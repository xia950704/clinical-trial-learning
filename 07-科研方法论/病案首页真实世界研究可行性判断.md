## 一、五个前置条件

核心不是"有没有数据"，而是这些数据能否回答一个具体临床问题。

| 条件 | 核心问题 | 详细说明 |
|------|---------|---------|
| **01 数据来源** | 字段来自病案编码库、首页上传系统、医保/DRG结算库，还是临床原始记录？需可追溯、可验证 | 至少回答：哪一级病案系统？时间范围？ICD版本？编码规则是否一致？是否去重？仅纳入首次住院？有无跨院数据链接？ |
| **02 变量完整性** | 暴露因素、研究结局、协变量是否全部收录于首页？缺失字段能否通过链接HIS/LIS/随访系统补充？ | 先画最小变量矩阵：暴露/分组、结局、混杂/协变量。三列中任何一列的关键字段缺失→题目降级、改题或补数据源 |
| **03 结局可靠性** | 院内死亡、住院天数等住院周期内结局较稳定；30天死亡、再入院等院外结局需链接外部数据源 | 院内结局（死亡、离院方式、住院天数、费用、是否转院、是否接受某类手术）记录路径清楚、可重复性高；院外结局需死亡登记、医保再入院记录、区域信息平台或随访数据链接 |
| **04 时间顺序** | 入院前合并症→住院中诊疗操作→出院结局的时间序列能否清晰排序？避免倒因为果 | 把住院全过程拆成清晰时间轴，先判断变量发生在"暴露前"还是"暴露后"。暴露后、结局前的混杂→可能是暴露导致的下游结果，强行调整会引入偏倚 |
| **05 混杂控制** | 年龄/性别/合并症可通过首页调整；但病情严重度、用药指征等关键混杂通常需额外补充数据 | 首页可控制：年龄、性别、入院途径、付费方式、年度、住院次数、部分合并症（通过ICD构建Charlson/Elixhauser指数）。首页无法控制：氧合水平、生命体征、实验室指标、影像严重程度、用药指征、医生治疗偏好 |

> **判定原则**：研究问题中最关键的暴露、结局、混杂因素，至少要能被**「可靠定义 + 可重复提取 + 可审计追溯」**。

## 二、变量完整性矩阵

### 首页稳定可获取的变量

- 人口学：年龄、性别
- 住院信息：住院次数、入院途径、住院天数、总费用
- 诊断：主要诊断、其他诊断（可构建Charlson/Elixhauser合并症指数）
- 操作：主要手术/操作
- 结局：离院方式

### 首页普遍缺失的变量

- 检验指标：血氧、肌酐、CRP、降钙素原
- 基线信息：生命体征、BMI、吸烟饮酒史
- 治疗细节：具体药物名称、剂量、给药时间、疗程

> 首页适合"编码明确、住院周期内可观察"的问题；不适合单独回答精细化治疗效果问题。

## 三、三个真实选题的可行性判断

### ✅ 可直接开展

**题目**：急性心肌梗死住院患者PCI使用与院内结局的真实世界研究

- 首页可准确定义：AMI主诊断、PCI操作、院内死亡/住院天数/费用
- 可风险调整：年龄、性别、合并症、医院层级/年度
- **结论表述**：宜用"关联"或"风险分层"，不轻易推导为治疗因果关系

### ⚠️ 可做，但需补充数据

**题目**：社区获得性肺炎患者ICU入住与院内死亡风险研究

- 首页可识别：肺炎诊断、ICU转入/转出、离院方式
- 缺失：CURB-65评分、氧合指标、炎症水平、影像病灶范围
- **需补充**：链接LIS、EMR系统获取病情严重度指标

### ❌ 不建议单独做

**题目**：某抗生素是否降低重症肺炎死亡率研究

- 首页缺失：药物名称、剂量、起始时间、疗程、换药原因
- 强指征混杂：治疗选择本身受病情严重度影响
- **结论**：仅靠首页数据很难支撑药物有效性的因果结论

## 四、实操清单：拿到首页数据后，先做这5步

1. **先定义题目**：明确研究对象、暴露/分组、结局
2. **再列变量矩阵**：检查暴露、结局、协变量是否都在首页里
3. **画时间轴**：确认暴露先于结局，混杂因素不会被暴露反过来影响
4. **评估混杂控制**：判断最关键的风险因素能否调整，不能调整时主动降级研究目标
5. **决定是否链接数据源**：如果关键变量不在首页，尽早考虑HIS、EMR、LIS、PACS、医保、随访或死亡登记

## 五、决策流程图

```
研究问题
    │
    ├── 暴露/结局/混杂能否在首页可靠定义？
    │       │
    │       ├── 是 → 可直接开展
    │       │         结论表述：关联/风险分层
    │       │
    │       └── 否 → 缺失变量能否通过链接HIS/LIS补充？
    │               │
    │               ├── 是 → 可做，但需补充数据
    │               │         结论表述：谨慎，注明数据局限
    │               │
    │               └── 否 → 不建议单独做
    │                         考虑其他数据源或研究设计
```

## 七、关键数字速查

| 指标 | 说明 |
|------|------|
| 首页稳定变量 | 年龄、性别、诊断、操作、住院天数、费用 |
| 首页缺失变量 | 检验指标、生命体征、药物细节 |
| 院内结局可靠性 | 高（死亡、住院天数） |
| 院外结局可靠性 | 低（需链接外部数据源） |
| 可调整的混杂 | 年龄、性别、合并症指数（Charlson/Elixhauser） |
| 需补充的混杂 | 病情严重度、用药指征、依从性 |
